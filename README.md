# Smart Appointment Management System

A full-stack web application for managing appointments efficiently using smart scheduling.

## Tech Stack
- **Frontend:** React.js + Vite + Tailwind CSS
- **Backend:** Node.js + Express.js
- **Database:** MongoDB + Mongoose
- **Auth:** JWT (JSON Web Tokens)

## Getting Started

### Prerequisites
- Node.js v18+
- MongoDB (local or Atlas)

### Installation
```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/smart-appointment-system.git
cd smart-appointment-system

# Install server dependencies
cd server && npm install

# Install client dependencies
cd ../client && npm install
```

### Running the App
```bash
# Terminal 1 - Backend
cd server && npm run dev

# Terminal 2 - Frontend
cd client && npm run dev
```

## Environment Variables
See `server/.env.example` for required variables.
